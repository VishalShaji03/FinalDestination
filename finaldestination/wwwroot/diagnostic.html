Quick Fixes

If you see 404 errors:
1. Make sure the API is running: dotnet run
2. Verify it's on port 5001: https://localhost:5001
3. Check Swagger UI loads: Open Swagger
If you see CORS errors:
The API has CORS enabled. Make sure you're running in Development mode.
If you see SSL errors:
Trust the development certificate: dotnet dev-certs https --trust
